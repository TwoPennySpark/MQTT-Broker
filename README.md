# MQTT Broker

Simple implementation of MQTT Broker v3.1.1:  
http://docs.oasis-open.org/mqtt/mqtt/v3.1.1/os/mqtt-v3.1.1-os.html  

Using c++17 and boost asio v1.79.0  
  
## Build & Run:  
```
git clone https://github.com/TwoPennySpark/MQTT-Broker.git  
cd MQTT-Broker/  
mkdir build  
cd build/  
cmake ..  
make install  
  
cd ../install  
./broker  
```
